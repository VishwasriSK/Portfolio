# How to Add Your Avatar Photo

## Quick Instructions:

1. **Save your photo as `avatar.jpg`**
   - Save your professional photo (the one you shared) as `avatar.jpg` in the Portfolio folder
   - Location: `c:\Users\Vichu's Victus\OneDrive\Desktop\Portfolio\avatar.jpg`

2. **Refresh the Portfolio Page**
   - Open the portfolio in your browser
   - Refresh the page (Ctrl + R or Cmd + R)
   - Your photo will now appear as a circular avatar above your name!

## File Details:

- **Photo File Name:** `avatar.jpg` (make sure it's exactly this name)
- **Photo Format:** JPG, PNG, or any common image format
- **Location:** Same folder as `index.html`
- **Size:** Any size works - the CSS will automatically crop it to a perfect circle

## What It Will Look Like:

The avatar will appear as a beautiful circular profile photo above your name in the hero section with:
- Teal border (4px)
- Smooth hover effect that scales up
- Professional shadow effect

## Color Palette Applied:

✅ Deep Navy Blue: #061e29 (Primary background)
✅ Teal Blue: #1d546d (Secondary backgrounds)
✅ Muted Teal: #5f9598 (Accents and borders)
✅ Light Cream: #f3f4f4 (Text and highlights)

This sophisticated palette creates an elegant, professional look that's perfect for interviewer portfolios!

## Troubleshooting:

- If the photo doesn't appear, make sure:
  1. The file is named exactly `avatar.jpg`
  2. It's in the same folder as `index.html`
  3. Your browser cache is cleared (Ctrl + Shift + Delete)
  4. The image file is not corrupted
